# Countries-CapitsLangs
Finding and mapping users' countries' capitals and their languages

## prerequisites
JDK 1.8  
Maven
## Installation

Build and run project with command line in windows:

```bash
  cd <project-path>
  mvn clean Install
  javac Application.java
  java -jar Application.java
```
